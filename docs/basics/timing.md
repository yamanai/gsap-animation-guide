# 时间控制

GSAP提供了丰富的时间控制功能，让你能够精确掌握动画的持续时间、延迟、重复等时间相关参数。本章将详细介绍如何使用这些功能。

## 基础时间参数

GSAP动画中最常用的时间参数包括：

```javascript
gsap.to(".box", {
  x: 200,
  duration: 2,    // 动画持续时间（秒）
  delay: 0.5,     // 动画开始前的延迟（秒）
  repeat: 3,      // 重复次数（不包括第一次）
  repeatDelay: 1, // 每次重复之间的延迟（秒）
  yoyo: true,     // 设置为true时，动画会来回播放
});
```
